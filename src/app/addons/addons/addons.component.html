How to Install My Chrome Extension

1. Download the Extension

Click the button below to download the Chrome extension file (.zip format):

Download Extension

Once downloaded, extract the contents of the zip file to a folder on your computer.

2. Load the Extension into Chrome

  1. Open Chrome and navigate to the Extensions page by entering chrome://extensions/ in the address bar.
  2. Enable Developer mode by toggling the switch in the upper-right corner of the page.
  3. Click the Load unpacked button.
  4. Select the folder where you extracted the extension files. Make sure it contains the manifest.json file.
  5. Once loaded, the extension will appear in your list of extensions. Ensure it is enabled.

3. Start Using the Extension

Once installed, the extension icon will appear in the Chrome toolbar. You can start using it immediately on supported websites.

  • Visit a webpage where you want to use the extension.
  • Click the extension icon in the toolbar to activate it.
  • Follow any on-screen prompts or instructions provided by the extension.
Note: If you encounter any issues during installation, make sure to check that the extension files are correctly extracted and that Developer mode is enabled in Chrome.
